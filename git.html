Commandes Description
git clone récupérer un projet distant
git add . ajouter du contenu
git commit -m valider/sauvegarder le contenu rajouté + commenter
git push envoyer ses fichiers sur le repo distant
git pull récupérer la dernière version de la branch
git fetch récupérer toutes les informations mises à jour du repo
git merge fusionner deux branches
git checkout créer une nouvelle branche/changer sur une branche déjà pré-créé
